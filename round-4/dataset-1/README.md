## Full Summary

This artifact provides the polity5_democratizers dataset: 1,046 country-year observations for 38 post-1990 democratizer countries (1990-2018) drawn from the Polity5 Project (p5v2018.xls). Post-1990 democratizers are identified via the Polity5 polity2 score: countries with polity2 ≤ 0 in any year 1985-1994 AND polity2 ≥ 6 in any year 1990-2010, with the first democratic year after 1989.

Each example (data row = one observation) has:
- input: JSON string of 12 Polity5 component scores: democ, autoc, xrreg, xrcomp, xropen, xconst, parreg, parcomp, exrec, exconst, polcomp, durable
- output: polity2 score as string (integer from -10 to +10)
- metadata_country, metadata_scode (3-letter code), metadata_year, metadata_period (5-year bin), metadata_ccode

Coverage: 38 countries, 1990-2018, 1,046 valid rows (polity2 in [-10, 10]). Average within-country SD of polity2 output = 3.11 (high variation supporting difference-in-differences identification). 5-year period bins: 1990-94, 1995-99, 2000-04, 2005-09, 2010-14, 2015-19.

Dataset choice rationale: polity5_democratizers was chosen over swiid_inequality because (1) democracy quality (polity2) is the direct outcome variable for the SDET triple-interaction DD experiment; (2) consistent 12-feature inputs vs SWIID's median of 2 features (64% of SWIID rows missing abs_red/rel_red); (3) higher within-country variation (SD=3.11 vs 1.63); (4) more countries (39 vs 36).

Data limitations: V-Dem sub-indices (v2jucomp, v2x_jucon, v2cseeorgs, v2csprtcpt, v2mecenefm) unavailable — uses polity2 as democracy proxy. NSTP/ILO social protection not included. Polity5 data ends at 2018 (p5v2018 version).

Output format follows exp_sel_data_out schema: {"datasets": [{"dataset": "polity5_democratizers", "examples": [...]}]}. Schema validation: PASSED.
